# IEEE-14-Bus-Power-Flow-with-Solar-PV-and-Battery-Storage
Modeled the IEEE 14-bus system in Python with pandapower. Added a 5 MW solar plant and an 8 MW battery, then measured how each shifted bus voltages and offloaded the slack bus. Visualized every scenario as a voltage profile against ±5% limits.  Tools: Python, pandapower, matplotlib
